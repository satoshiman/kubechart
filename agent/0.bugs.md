### [FIXED] String.repeat() negative count error when metrics ~100%

- **Location**: `src/metrics/formatter.ts:90`
- **Error**: `Invalid count value: -7657` / `-757`
- **Root Cause**: When cluster overloaded (percent > 100%), `filled` exceeds `width`, causing `empty = width - filled` to be negative
- **Fix**: Clamp `filled` with `Math.min(width, ...)` to ensure `empty` is always >= 0
